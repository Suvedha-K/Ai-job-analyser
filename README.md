AI Job Market Analysis

Project Overview

This project analyzes AI job market data using Excel and SQL.

Tools Used

- Excel
- SQLite (DB Browser for SQLite)
- SQL
- GitHub

Dataset

- ai_job_market_insights.csv

Work Done

- Cleaned the dataset in Excel.
- Created Pivot Tables.
- Created charts for analysis.
- Imported the dataset into SQLite.
- Performed basic SQL exploration using SELECT and COUNT queries.

Project Structure

- dataset/: Contains raw dataset.
- database/: Contains SQLite database.
- excel_analysis/: Contains pivot tables and charts.
- sql_queries/: SQL scripts used for analysis.
- insights.md: Findings and conclusions.

Goal

To analyze job roles, required skills, and automation trends in the AI job market.
