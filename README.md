# 📊 AI Job Market Analysis

## 📌 Project Overview

This project analyzes AI job market data using Excel and SQL. The objective is to explore job roles, required skills, and automation trends through data cleaning, visualization, and SQL-based analysis.

---

## 🛠️ Tools Used

- Microsoft Excel
- SQLite (DB Browser for SQLite)
- SQL
- GitHub

---

## 📂 Dataset

- **File:** `ai_job_market_insights.csv`
- Contains information about AI-related job roles, required skills, salary levels, and automation risk.

---

## 📈 Work Done

- Cleaned the dataset using Excel.
- Created Pivot Tables for analysis.
- Built charts and visualizations.
- Imported the dataset into SQLite.
- Created a table named `job_markets_utf8`.
- Executed SQL queries for exploratory analysis.
- Organized the project into a professional GitHub repository.
